# 🌌 AI Story Generator

An AI-powered desktop application built using **Python**, **Tkinter**, and **Ollama (Llama 3.2:1B)** that generates emotional stories based on a user-provided theme. The application features a modern graphical interface, animated loading indicator, and smooth typewriter-style story display.

---

## 📖 Project Overview

The AI Story Generator enables users to create engaging stories by simply entering a topic or theme. It leverages the **Ollama Llama 3.2:1B** language model to generate detailed stories with a structured narrative including a beginning, conflict, climax, and conclusion.

---

## ✨ Features

- 🌌 AI-powered story generation using Ollama
- 📝 Generates stories of approximately **1000–1500 words**
- 🎭 Emotion-driven storytelling
- 🎬 Structured story flow with beginning, conflict, climax, and ending
- 🖥️ Modern desktop GUI built with Tkinter
- ⚡ Animated loading indicator during story generation
- ⌨️ User-friendly theme input
- 🎯 Smooth typewriter animation for displaying generated stories
- 🌙 Dark-themed interface for improved readability
- 🔄 Multi-threaded processing to keep the interface responsive

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/hemasree200807/AI-Story-Generator.git
```

### Navigate to the project folder

```bash
cd AI-Story-Generator
```

### Install Ollama

Download and install Ollama from:

https://ollama.com/download

### Install the required package

```bash
pip install ollama
```

### Download the AI model

```bash
ollama pull llama3.2:1b
```

### Start Ollama

```bash
ollama serve
```

### Run the application

```bash
python app.py
```

---

## ⚙️ How It Works

1. Launch the application.
2. Enter a story topic or theme.
3. Click **Generate Story**.
4. The application sends the prompt to the Ollama Llama 3.2:1B model.
5. A loading animation is displayed while the AI generates the story.
6. The generated story appears with a smooth typewriter animation.
